# football-transfermarkt-internship
## Domain: Sports Analytics (Football)

### Why this domain?
Football is a data-rich sport where player performance, transfers, and market value
can be analyzed to understand trends, team strategies, and financial decisions.

### Problem Statement
Using Transfermarkt data, we aim to analyze:
- Player market value trends
- Transfer spending patterns
- Factors influencing player valuation

This analysis can help clubs, scouts, and analysts make better transfer decisions.
